To show the information of a single bank account in an XML file enter the following url in your browser:
http://localhost:8082/Netprog_Project/rest/bankAccountService/bankaccountxml/{accountId}*

To show the information of a single bank account in an JSON file enter the following url in your browser:
http://localhost:8082/Netprog_Project/rest/bankAccountService/bankaccountjson/{accountId}*

* In place of accountId enter the number of the account you want to see information on.
